## To print the tree structure 
- make sure you have wsl installed (Windows subsystem for linux)
- ```ls -R | grep ":$" | sed -e 's/:$//' -e 's/[^-][^\/]*\//--/g' -e 's/^/   /' -e 's/-/|/'```
